# Customer Application Roadmap

## 📱 Core Pages

### 1. Home / Explore Page (✅ In Progress)
- **Hero Section**: Immersive search bar with location, dates, and guest selection.
- **Categories Bar**: Sticky navigation for filtering by property type (e.g., Pools, Farms, Views).
- **Featured Listings**: Responsive grid of high-quality farmhouse cards.
- **Redesign Status**: Modern glassmorphism navbar and premium card styles implemented.

### 2. Property Details Page (🚧 Next Priority)
- **Image Gallery**: High-resolution grid layout for property photos.
- **Property Info**: Detailed description, sleeping arrangements, and house rules.
- **Amenities List**: Comprehensive list of what's offered (Wifi, Pool, AC, etc.).
- **Host Profile**: Brief bio and rating of the host.
- **Reviews Section**: User ratings and written feedback.
- **Booking Widget**: Sticky sidebar for selecting dates and viewing price breakdown.

### 3. Search Results Page
- **Advanced Filters**: Price range slider, amenities checkboxes, instant book toggle.
- **Map View**: Interactive map showing property locations and prices.
- **List View**: Vertical list of available properties matching criteria.

### 4. Checkout & Booking
- **Review Itinerary**: Confirm dates, guests, and total price.
- **Payment Gateway**: Secure integration (Razorpay/Stripe) for credit cards, UPI, etc.
- **Confirmation**: Success page with booking reference and next steps.

### 5. User Dashboard (My Trips)
- **Upcoming Trips**: Details for future bookings with "Get Directions" and "Message Host" options.
- **Past Trips**: History of completed stays with an option to leave a review.
- **Cancelled**: Status of cancelled reservations.

### 6. Wishlists
- **Saved Properties**: Grid view of farmhouses marked as favorites.
- **Collections**: Ability to organize favorites into folders (e.g., "Summer Vacation", "Birthday Trip").

### 7. Profile & Settings
- **Personal Info**: Edit name, email, phone number, and profile picture.
- **Security**: Password management and login history.
- **Payments**: Manage saved cards and billing history.

### 8. Inbox / Messages
- **Chat Interface**: Real-time messaging with hosts for pre-booking questions and stay coordination.
- **Notifications**: Alerts for booking status updates and new messages.

### 9. Authentication
- **Login**: Email/Password and Google OAuth.
- **Register**: New user sign-up flow.
- **Forgot Password**: Email recovery flow.

---

## 🌟 Key Features

- **Smart Search**: "I'm Flexible" dates and destination suggestions.
- **Real-time Availability**: Prevents double bookings.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Dark Mode**: Optional dark theme for better viewing at night.
- **Loading Skeletons**: Smooth loading states for better UX.
- **Toast Notifications**: Non-intrusive alerts for actions (e.g., "Added to wishlist").
